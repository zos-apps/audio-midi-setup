## About

Configure audio devices and MIDI settings for your system.

## Features

- Audio device configuration
- MIDI device setup
- Sample rate settings
- Buffer size control

## Installation

### From App Store
Open the **App Store** in zOS and search for "Audio MIDI Setup".

### Via npm
```bash
npm install @zos-apps/audio-midi-setup
```

## Usage

- Select audio device from list
- Configure sample rate
- Set up MIDI inputs/outputs



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
